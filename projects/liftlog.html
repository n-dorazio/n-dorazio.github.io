    • LiftLog

    Fitness Tracking Application

    Human Computer Interaction

    LiftLog is a frontend fitness tracking application designed for a Human Computer Interaction course, focusing on usability and user experience. The project explores how intuitive design can enhance user engagement in fitness applications.

    The application allows users to track workouts, monitor progress, and maintain fitness goals through a carefully designed interface that prioritizes ease of use and accessibility.

    View Source Code

    Project Goals

    Usability Research

    Conduct comprehensive usability testing to identify pain points in fitness tracking applications and design solutions that address user needs.

    User-Centered Design

    Create an intuitive interface that makes fitness tracking accessible to users of all experience levels.

    Academic Application

    Apply Human Computer Interaction principles to real-world design challenges and document the process.

    Design Process

    1

    Research & Analysis

    Analyzed existing fitness apps to understand common usability issues and user pain points.

    2

    User Research

    Conducted interviews and surveys with potential users to understand their needs and preferences.

    3

    Design & Prototyping

    Created wireframes and prototypes focusing on usability and intuitive navigation.

    4

    Usability Testing

    Tested the application with users and iterated based on feedback and observations.

    Key Features

    Workout Tracking

    Easy-to-use interface for logging exercises, sets, and reps with minimal cognitive load.

    Progress Monitoring

    Visual progress indicators and charts to help users track their fitness journey.

    Goal Setting

    Intuitive goal creation and tracking system with clear feedback mechanisms.

    User Profiles

    Personalized experience with customizable user profiles and preferences.
